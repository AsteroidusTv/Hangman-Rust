# Hangman-Rust

## How to use Hangman-Rust : 

### Run with the executable

Download the release and run it
```
./main
```
### Run with cargo

Clone the repository
```
git clone https://github.com/AsteroidusTv/Hangman-Rust.git
```
Go into the folder
```
cd Hangman-Rust 
```
Run it
```
cargo run 
```

## Good luck !
